#CSS-LESS-SASS

####CSS

###### Selectors

* Universal Selector

  <dl>
      <dd>* {</dd>
      <dd> color: green;</dd>
      <dd> font-size: 20px;</dd>
      <dd> line-height: 25px;</dd>
      <dd>}</dd>
  </dl>


* Element type Selector

  <dl>
      <dd>p {</dd>
      <dd> font-size: 20px;</dd>
      <dd>}</dd>
  </dl>

* ID Selector

  <dl>
      <dd>#container {</dd>
      <dd> width: 960px;</dd>
      <dd>  margin: 0 auto;</dd>
      <dd>}</dd>
  </dl>

* Class Selector

  <dl>
      <dd>.box {</dd>
      <dd> padding: 20px;</dd>
      <dd> margin: 10px;</dd>
      <dd> width: 240px;</dd>
      <dd>}</dd>
  </dl>

* Descendant Combinator

  <dl>
      <dd>#container .box {</dd>
      <dd> padding: 20px;</dd>
      <dd>}</dd>
  </dl>

* Child Combinator

  <dl>
      <dd>#container > .box {</dd>
      <dd> float: left;</dd>
      <dd>}</dd>
  </dl>

* General Sibling Combinator

  <dl>
      <dd>h2 ~ p {</dd>
      <dd> margin-bottom: 20px;</dd>
      <dd>}</dd>
  </dl>

* Adjacent Sibling Combinator

  <dl>
      <dd>p + p {</dd>
      <dd> margin-bottom: 20px;</dd>
      <dd>}</dd>
  </dl>

* Attribute Selector

  <dl>
      <dd>input[type] {</dd>
      <dd> width: 200px;</dd>
      <dd>}</dd>
  </dl>

* Pseudo-class

  <dl>
      <dd>a:hover {</dd>
      <dd> color: red;</dd>
      <dd>}</dd>
  </dl>

* Pseudo-element

  <dl>
      <dd>.container:before {</dd>
      <dd> width: 200px;</dd>
      <dd>}</dd>
  </dl>
